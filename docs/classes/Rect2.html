Class Rect2

Color metatype, wrapper for godot_rect2.

Construct using the idiom Rect2(...), which calls __new.

Components may be accessed through elements, individually using x/y/width/height or as Vector2 position/size:

typedef union godot_rect2 {
    float elements[4];
    struct { float x, y, width, height; };
    struct { Vector2 position; Vector2 size; };
} godot_rect2;

Rect2:clip (b) Returns the intersection of this Rect2 and b.

Rect2:has_no_area () Returns true if the Rect2 is flat or empty.
