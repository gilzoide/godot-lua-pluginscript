Module string_extras

Extra functionality for Lua's string library.

Functions

string.join (sep, ...)
Returns a Lua string with all values joined by separator. tostring is called to each of the passed values before joining.

Parameters:

  • sep string
  • ... Values to be joined, stringified by tostring.

Returns:

    string

Usage:

    assert(string.join(',', 1, 2, 'three', Array()) == '1,2,three,[]')
string.replace (str, pattern, replacement)
Performs plain substring substitution, with no characters in pattern or replacement being considered magic.

Parameters:

  • str string
  • pattern string
  • replacement string

Returns:

    string

See also:

    string.gsub

Usage:

    assert(string.replace('Dot.percent.arent.magic', '.', '%') == 'Dot%percent%arent%magic')
